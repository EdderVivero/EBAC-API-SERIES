# 🦆 El Pato Cinéfilo

Una aplicación web moderna para explorar información sobre series y actores, impulsada por la API de **TV Maze**.

## 🚀 Características

- **Buscador Inteligente**: Encuentra series y actores en una sola búsqueda.
- **Top 10 Dinámico**: Acceso rápido a las mejores series y actores populares.
- **Diseño Responsivo**: Experiencia optimizada para móviles con menú de hamburguesa.
- **Detalles Completos**: Información sobre el país, nacimiento, género y resúmenes de series.
- **Arquitectura Limpia**: Estilos organizados mediante SASS modular.

## 🛠️ Tecnologías

- **HTML5** & **CSS3 (SASS/SCSS)**
- **JavaScript (Vanilla)**
- **Axios** (para peticiones a la API)
- **TV Maze API**

## 📦 Instalación

1. Clona este repositorio.
2. Abre el archivo `index.html` en tu navegador favorito.
3. ¡Disfruta explorando!
